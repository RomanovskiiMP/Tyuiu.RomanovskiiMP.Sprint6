# Задание Sprint 6 Task 1 Variant 22

## Описание задания
Написать программу, которая выводит таблицу значений функции:
**F(x) = 3x + 2 − (2x − x) / (cos(x) + 1)**

Произвести табулирование функции f(x) на заданном диапазоне [-5; 5] с шагом 1. 
Выполнить проверку деления на ноль. При делении на ноль вернуть значение 0.

## Структура проекта

### 1. Tyuiu.RomanovskiiMP.Sprint6.Task1.V22.Lib
Библиотека классов с методами:
- `GetMassFunction(double x)` - вычисляет значение функции для одного x
- `GetMassFunction(int startValue, int stopValue)` - табулирует функцию на диапазоне

### 2. Tyuiu.RomanovskiiMP.Sprint6.Task1.V22.Test  
Модуль тестирования с unit-тестами для проверки корректности вычислений и табулирования

### 3. Tyuiu.RomanovskiiMP.Sprint6.Task1.V22
Основное WinForms приложение с графическим интерфейсом для ввода диапазона и вывода таблицы

## Интерфейс приложения
- **GroupBox "Условие"** - описание задачи и формула функции
- **GroupBox "Ввод данных"** - поля для ввода начального и конечного значения диапазона
- **GroupBox "Вывод данных"** - таблица результатов в формате:
  ```
  +----------+----------+
  |    X     |   f(x)   |
  +----------+----------+
  |   -5     |   -12.50 |
  |   -4     |    -8.93 |
  |   ...    |    ...   |
  +----------+----------+
  ```
- **Кнопка "Выполнить"** - запуск табулирования
- **Кнопка "Справка"** - информация об авторе

## Особенности реализации
- Проверка деления на ноль: если cos(x) + 1 ≈ 0, возвращается 0
- Результаты округляются до 2 знаков после запятой
- Упрощение формулы: (2x - x) = x
- Таблица выводится в моноширинном шрифте для корректного отображения
