# Задание Sprint 6 Task 2 Variant 8

## Описание задания
Написать программу, которая выводит таблицу значений функции:
**F(x) = cos(2x) + sin(x) / (x + 2,5) + 2x**

Произвести табулирование функции f(x) на заданном диапазоне [-5; 5] с шагом 1.
Выполнить проверку деления на ноль. При делении на ноль вернуть значение 0.
Значения занести в DataGridView. Значения округлить до двух знаков после запятой.

## Структура проекта

### 1. Tyuiu.RomanovskiiMP.Sprint6.Task2.V8.Lib
Библиотека классов с методами:
- `GetMassFunction(double x)` - вычисляет значение функции для одного x
- `GetMassFunction(int startValue, int stopValue)` - табулирует функцию на диапазоне

### 2. Tyuiu.RomanovskiiMP.Sprint6.Task2.V8.Test  
Модуль тестирования с unit-тестами:
- Тест табулирования массива значений
- Тест вычисления одного значения
- Тест проверки деления на ноль (x = -2.5)

### 3. Tyuiu.RomanovskiiMP.Sprint6.Task2.V8
Основное WinForms приложение с графическим интерфейсом и DataGridView

## Интерфейс приложения
- **GroupBox "Условие"** - описание задачи и формула функции
- **GroupBox "Ввод данных"** - поля для ввода начального и конечного значения диапазона
- **GroupBox "Вывод данных"** - DataGridView с колонками:
  - **X** - значения аргумента
  - **F(X)** - значения функции
- **Кнопка "Выполнить"** - запуск табулирования с эффектами при наведении
- **Кнопка "Справка"** - информация об авторе
- **ToolTip** - подсказки при наведении на элементы

## Особенности реализации
- Проверка деления на ноль: если x + 2.5 ≈ 0, возвращается 0
- Результаты округляются до 2 знаков после запятой
- DataGridView автоматически очищается перед новым расчетом
- Интерактивные эффекты кнопок (изменение цвета при наведении)
- Обработка ошибок ввода с выводом сообщений

## Математическая функция
F(x) = cos(2x) + sin(x) / (x + 2,5) + 2x

Где:
- cos(2x) - косинус удвоенного аргумента
- sin(x) / (x + 2,5) - синус x, деленный на (x + 2,5)
- 2x - удвоенный аргумент
