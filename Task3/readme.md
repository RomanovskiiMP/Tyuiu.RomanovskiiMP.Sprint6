# Задание Sprint 6 Task 3 Variant 18

## Описание задания
Дан массив 5 на 5 элементов. Заменить чётные значения в четвёртой строке на 0. Результат вывести в DataGridView.

**Исходный массив:**
```
-19 -19   1  18  75
  3  -4  -6 -12 -15
  6   2   2 -14  -9
-10  15  -5   6 -13  ← четвертая строка
-15  -9   7   1   0
```

**Результат (четные значения в 4-й строке заменены на 0):**
```
-19 -19   1  18  75
  3  -4  -6 -12 -15
  6   2   2 -14  -9
  0  15  -5   0 -13  ← -10 и 6 заменены на 0
-15  -9   7   1   0
```

## Структура проекта

### 1. Tyuiu.RomanovskiiMP.Sprint6.Task3.V18.Lib
Библиотека классов с методами:
- `Calculate(int[,] matrix)` - заменяет четные значения в четвертой строке на 0
- `GetMatrix()` - возвращает исходный массив из задания

### 2. Tyuiu.RomanovskiiMP.Sprint6.Task3.V18.Test  
Модуль тестирования с 3 unit-тестами:
- Тест полного преобразования массива
- Тест получения исходного массива
- Тест конкретных изменений в четвертой строке

### 3. Tyuiu.RomanovskiiMP.Sprint6.Task3.V18
Основное WinForms приложение с двумя DataGridView для сравнения

## Интерфейс приложения
- **GroupBox "Условие"** - описание задачи
- **GroupBox "Исходный массив"** - DataGridView с исходными данными
  - Четвертая строка выделена синим цветом
- **GroupBox "Результат"** - DataGridView с результатом обработки
  - Четвертая строка выделена синим цветом
  - Измененные значения выделены зеленым цветом
- **Кнопка "Выполнить"** - запуск обработки массива
- **Кнопка "Справка"** - информация об авторе

## Особенности реализации
- Автоматическая загрузка исходного массива при запуске
- Визуальное выделение четвертой строки и измененных значений
- DataGridView настроены как таблицы 5x5 без заголовков
- Размеры ячеек оптимизированы для отображения чисел
- Обработка ошибок с выводом сообщений

## Алгоритм работы
1. При загрузке формы отображается исходный массив
2. Четвертая строка выделяется синим цветом
3. При нажатии "Выполнить":
   - Создается копия исходного массива
   - В четвертой строке (индекс 3) проверяется каждый элемент
   - Четные элементы (делящиеся на 2 без остатка) заменяются на 0
   - Результат отображается во втором DataGridView
   - Измененные значения выделяются зеленым цветом

## Результат работы
В четвертой строке исходного массива [-10, 15, -5, 6, -13]:
- **-10** (четное) → **0**
- **15** (нечетное) → **15** (без изменений)
- **-5** (нечетное) → **-5** (без изменений)  
- **6** (четное) → **0**
- **-13** (нечетное) → **-13** (без изменений)

Итоговая четвертая строка: [0, 15, -5, 0, -13]
