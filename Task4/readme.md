# Задание Sprint 6 Task 4 Variant 9

## Описание задания
Написать программу, которая выводит таблицу значений функции:
**F(x) = (2*x - 3) / (cos(x) - 2*x) + 5*x - sin(x)**

Произвести табулирование функции f(x) на заданном диапазоне [-5; 5] с шагом 1.
Произвести проверку деления на ноль. При делении на ноль вернуть значение 0.
Округлить результат до двух знаков после запятой.
Результат вывести в TextBox, построить график функции и сохранить в файл OutPutFileTask4V9.txt по нажатию кнопки.

## Структура проекта

### 1. Tyuiu.RomanovskiiMP.Sprint6.Task4.V9.Lib
Библиотека классов с методами:
- `GetMassFunction(double x)` - вычисляет значение функции для одного x
- `GetMassFunction(int startValue, int stopValue)` - табулирует функцию на диапазоне
- `SaveToFile(int startValue, int stopValue, string filePath)` - сохраняет результаты в файл

### 2. Tyuiu.RomanovskiiMP.Sprint6.Task4.V9.Test  
Модуль тестирования с 3 unit-тестами:
- Тест табулирования массива значений
- Тест вычисления одного значения
- Тест сохранения в файл

### 3. Tyuiu.RomanovskiiMP.Sprint6.Task4.V9
Основное WinForms приложение с TextBox, графиком и функциями сохранения

## Интерфейс приложения
- **GroupBox "Условие"** - описание задачи и формула функции
- **GroupBox "Ввод данных"** - поля для ввода начального и конечного значения диапазона
- **GroupBox "Вывод данных"** - TextBox с результатами табулирования в формате:
  ```
  X = -5    F(X) = 12.34
  X = -4    F(X) = 8.76
  ...
  ```
- **GroupBox "График функции"** - Chart с визуализацией функции
- **Кнопка "Выполнить"** - запуск табулирования и построения графика
- **Кнопка "Сохранить"** - сохранение результатов в файл OutPutFileTask4V9.txt
- **Кнопка "Справка"** - информация об авторе

## Особенности реализации
- Проверка деления на ноль: если cos(x) - 2*x ≈ 0, возвращается 0
- Результаты округляются до 2 знаков после запятой
- График строится с использованием System.Windows.Forms.DataVisualization.Charting
- Тип графика: Spline (сглаженная кривая)
- Автоматическая настройка осей и сетки
- Сохранение в файл в формате: "X\tF(X)" (разделитель - табуляция)

## Математическая функция
F(x) = (2*x - 3) / (cos(x) - 2*x) + 5*x - sin(x)

Где:
- (2*x - 3) / (cos(x) - 2*x) - дробная часть с проверкой деления на ноль
- 5*x - линейная составляющая
- sin(x) - синусоидальная составляющая

## Функциональность
1. **Табулирование**: Вычисление значений функции на заданном диапазоне
2. **Визуализация**: Построение графика функции с настройкой осей
3. **Сохранение**: Экспорт результатов в текстовый файл
4. **Обработка ошибок**: Проверка деления на ноль и некорректного ввода

## Файл результатов

```
-4	8.76

...
```
