# Task 6 - Поиск слов с буквой 'v'

## Задание
Сгенерируй файл InPutFileTask6V10.txt с любым текстом.
Загрузить файл в textBoxIn через openFileDialog.
Вывести слова, в которых встречается v, в результирующую строку и вывести ее в textBoxOut.

## Структура проекта
- **Tyuiu.RomanovskiiMP.Sprint6.Task6.V10.Lib** - библиотека с логикой обработки текста
- **Tyuiu.RomanovskiiMP.Sprint6.Task6.V10.Test** - модульные тесты
- **Tyuiu.RomanovskiiMP.Sprint6.Task6.V10** - WinForms приложение

## Функциональность
- Загрузка текстового файла через OpenFileDialog
- Отображение содержимого файла в textBoxIn
- Поиск и извлечение слов, содержащих букву 'v'
- Вывод результата в textBoxOut
